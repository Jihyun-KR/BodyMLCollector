# 🤖 Body Type ML Data Collector

> 웹캠 하나로 실시간 신체 좌표를 뽑아 Azure ML에 바로 올릴 수 있는 CSV를 자동 생성하는 브라우저 기반 수집 툴

**🔗 라이브 데모:** https://jihyun-kr.github.io/BodyMLCollector/

---

## ✨ 왜 만들었나요?

체형 분류 ML 모델을 만들려면 정확한 신체 좌표 데이터가 필요한데, 3D 스캐너 없이 웹캠 하나로 수집할 수 있어야 했어요.
수집과 동시에 피처 추출, CSV 저장까지 한 번에 — 후속 ML 파이프라인이 바로 쓸 수 있는 형태로 만들었습니다.

---

## 🚀 바로 쓰기

설치 없이 링크 하나로 실행돼요.

👉 **https://jihyun-kr.github.io/BodyMLCollector/**

1. 동의 모달 확인
2. 카메라 시작 클릭
3. 전신이 보이도록 서기
4. 자동 카운트다운 후 캡처
5. CSV 다운로드

---

## 🛠️ 주요 기능

| 기능 | 설명 |
|------|------|
| 실시간 랜드마크 | MediaPipe Pose 33개 관절 실시간 추출 및 스켈레톤 시각화 |
| 자동 캡처 | 자세 안정화 감지 → 5초 카운트다운 후 자동 캡처 |
| 줌 슬라이더 | 0.3×~1.5× 조절 — 카메라 거리 상관없이 전신 촬영 가능 |
| 37개 피처 자동 계산 | 거리 19개 + 관절 각도 12개 + 비율 6개 |
| CSV 3종 출력 | 랜드마크 / 피처 / 통합 ML용 — Azure ML 바로 사용 가능 |
| 개인정보 보호 | 얼굴 랜드마크 CSV 저장 제외, 서버 전송 없음, 동의 팝업 |

---

## 📊 수집 데이터 구조

```
📍 랜드마크 좌표    — 33개 × (2D + World 3D)
📏 거리 피처        — 어깨폭, 골반폭, 상체길이, 허리너비 등 19개
📐 관절 각도        — 어깨, 팔꿈치, 무릎, 척추 기울기 등 12개
⚖️ 비율 피처        — 어깨/골반비, WHR, 상체/하체 비율 등 6개
```

---

## 🔒 개인정보 보호 설계

- 개인정보보호법 제15조 기반 동의 모달
- 얼굴 식별 정보 (눈·코·입·귀) CSV 저장 제외
- 카메라 영상 브라우저 내 처리만 — 서버 전송 없음
- 좌표값만 저장, 이미지 미저장

---

## 🏗️ 팀 파이프라인에서의 위치

```
[본인] 웹캠 수집툴 → CSV 출력
    → 픽셀→cm 변환 (RandomForest 보정)
    → 체형 분류 (GMM, 5가지 체형)
    → 스타일 추천
    → UI 표시 (Figma 서비스)
```
