Problem

왜 만들었나요?

📐
정확한 신체 좌표 필요
체형 분류 ML 모델 학습을 위한 정량적 신체 데이터가 필요했어요
💸
별도 장비 없이
3D 스캐너 없이 웹캠 하나로 수집 가능해야 했어요. 수동 측정은 오차와 확장성 문제가 있었어요
⚡
즉시 ML에 활용
수집과 동시에 피처 추출, CSV 저장까지 한 번에 — 후속 파트가 바로 쓸 수 있는 형태로
Solution

어떻게 구현했나요?

  • 01브라우저에서 바로 실행 — 설치 없이 HTML 파일 하나로 동작 GitHub Pages 배포
  • 02MediaPipe Pose로 실시간 33개 랜드마크 추출 및 스켈레톤 시각화
  • 03자세 안정화 감지 → 자동 5초 카운트다운 캡처 움직임 diff 기반
  • 04줌 슬라이더 (0.3×~1.5×) — 카메라 거리와 상관없이 전신 촬영 가능
  • 05Azure ML 바로 사용 가능한 Wide-format CSV 3종 다운로드 37개 피처 자동 계산

개인정보 보호 설계

🔒 개인정보보호법 대응

영상은 브라우저 내 처리만 — 서버 전송 없음
좌표값만 저장 — 이미지 미저장, 삭제권 보장
⚖️ AI 윤리 준수

추천 결과 — 특정 체형 비하 없는 중립 표현
